# ERP Print Shop (New Clean Start)

مشروع جديد بالكامل لإدارة المطابع الورقية، منفصل عن المشروع السابق.

## المميزات (النسخة الأولى)
- تسجيل العملاء
- إنشاء أوامر الطباعة
- تتبع حالة الطلب
- لوحة إحصائيات بسيطة

## التشغيل

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

ثم افتح:
`http://127.0.0.1:5000`

## ملاحظات
- قاعدة البيانات SQLite داخل الملف: `erp_new.db`
- هذا الإصدار بداية نظيفة، ويمكن توسيعه لاحقًا (مخزون، فواتير، موظفين، صلاحيات...)
